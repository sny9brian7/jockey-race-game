# 🏇 ジョッキー視点レースゲーム

騎手の一人称視点で名馬に騎乗し、歴史的レースを追体験する3D競馬レースゲームです。
ブラウザだけで動きます（ビルド不要・Three.js使用）。

## 遊び方

`index.html` をブラウザで開くだけです。

👉 GitHub Pages で公開している場合はそのままブラウザでプレイできます。

## 収録レース（12頭立て・相手馬も史実の出走馬）

| レース | コース | 騎乗馬 |
|---|---|---|
| 1997 天皇賞（春） | 京都 芝3200m（外回り・右回り） | マヤノトップガン |
| 1999 有馬記念 | 中山 芝2500m（内回り・右回り） | グラスワンダー |
| 2022 天皇賞（秋） | 東京 芝2000m（左回り） | イクイノックス |
| 2005 日本ダービー | 東京 芝2400m（左回り） | ディープインパクト |

コースは各競馬場の実測値（一周距離・直線長・回り方向）を近似再現しています。

## 操作

| キー | 動作 |
|---|---|
| ↑ / W | 追う（加速・スタミナ消費） |
| ↓ / S | 抑える（消耗を抑える。ただし引っ張ると掛かる） |
| ← → / A・D | 進路変更 |
| SPACE | ムチ（一時加速・スタミナ消費） |
| V | 後方確認（押している間） |

## ゲームの要点

- **スタミナ**: 回復なし。スリップストリーム（前の馬の4〜9m後ろ）で消耗-40%
- **折り合い**: 序盤450mの位置取りはフリー。それ以降に追う・引っ張る・前が開く・詰まると「掛かって」消耗が激増。壁の後ろで我慢させるのが唯一の対処
- **ペース**: 先頭の1000m通過でハイ/ミドル/スローを判定。ハイは前崩れ、スローは前残り
- **かなり難しい**です。まずタイトル画面の「勝ち方デモ」を見るのがおすすめ
